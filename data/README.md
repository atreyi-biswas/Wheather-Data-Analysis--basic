# Data (data)

## Purpose
This folder stores the final processed dataset generated from the weather data pipeline.

## Contents
- weather_report.csv → Clean structured dataset containing:
  - City-wise weather data
  - Temperature, humidity, wind speed
  - Weather conditions
  - Timestamps

## Role in Project
This is the output layer of the project. The data stored here is used for:
- Data analysis
- Visualization (Power BI / Python charts)
- Reporting and insights
